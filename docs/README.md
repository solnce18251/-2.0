# 🌐 GitHub Pages Инструкция

## Как включить GitHub Pages

### Шаг 1: Откройте настройки репозитория

Перейдите на вкладку **Settings**:
```
https://github.com/solnce18251/-2.0/settings/pages
```

### Шаг 2: Настройте Pages

**Build and deployment:**

| Параметр | Значение |
|----------|----------|
| **Source** | Deploy from a branch |
| **Branch** | `main` |
| **Folder** | `/docs` |
| **Root** | `/ (root)` |

### Шаг 3: Сохраните

Нажмите кнопку **Save**

### Шаг 4: Подождите 1-2 минуты

GitHub обработает файлы и опубликует сайт.

## ⚠️ Важно

### Демо-режим

Эта версия сайта работает в **демо-режиме**:
- 🎭 Показаны примерные данные
- 📊 Графики и статистика — моковые
- 🔄 Парсинг и импорт не работают

### Для полноценной работы:

1. **Локальный запуск:**
   ```bash
   python run.py server
   ```

2. **Развёртывание на Render:**
   - Следуйте инструкции в `DEPLOY_RENDER.md`
   - Ссылка: https://render.com/deploy?repo=https://github.com/solnce18251/-2.0

## Обновление статической версии

После изменений в проекте:

```bash
python generate_static.py
git add docs
git commit -m "docs: update static version"
git push
```

GitHub Pages автоматически обновится через 1-2 минуты.

## Troubleshooting

### Сайт не обновляется

1. Проверьте что файлы в папке `/docs` закоммичены
2. Подождите 2-3 минуты
3. Очистите кэш браузера (Ctrl+Shift+R)

### 404 ошибка

1. Убедитесь что выбран правильный branch (`main`)
2. Убедитесь что выбрана папка `/docs`
3. Проверьте что `index.html` существует

### Не работают стили

1. Проверьте консоль браузера (F12)
2. Убедитесь что CDN доступны
3. Попробуйте другой браузер
