# 📥 Импорт данных от HR-аналитика

## Инструкция по загрузке резюме из Excel/CSV файлов

### Способ 1: Через веб-интерфейс (рекомендуется)

1. **Откройте сайт:** http://127.0.0.1:5000
2. **Нажмите кнопку** "Импорт резюме"
3. **Выберите файл** Excel (.xlsx, .xls) или CSV
4. **Нажмите** "Загрузить"
5. **Дождитесь** завершения импорта

---

### Способ 2: Через командную строку

```bash
cd "c:\Users\solnc\Desktop\Парсер 2.0"
python import_resumes.py <путь_к_файлу> [источник]
```

**Пример:**
```bash
python import_resumes.py "C:\Users\solnc\Downloads\resumes.xlsx" hr_analyst
```

## Требования к файлу

### Поддерживаемые форматы:
- ✅ `.xlsx` (Excel 2007+)
- ✅ `.xls` (Excel 97-2003)
- ✅ `.csv` (CSV с запятыми или точкой с запятой)

### Рекомендуемые колонки:

| Колонка | Описание | Пример |
|---------|----------|--------|
| **Должность** | Название позиции | Python разработчик |
| **Зарплата от** | Минимальная зарплата | 150000 |
| **Зарплата до** | Максимальная зарплата | 250000 |
| **Город** | Город кандидата | Москва |
| **Опыт работы (лет)** | Лет опыта | 3 |
| **Уровень** | Уровень специалиста | Middle |
| **Навыки** | Ключевые навыки | Python, Django, SQL |
| **Дата публикации** | Дата размещения | 2024-01-15 |

## Обработка ошибок

### Файл не загружается
- Проверьте формат файла (.xlsx, .xls, .csv)
- Убедитесь что размер файла < 16MB
- Проверьте что файл не повреждён

### Данные не импортируются
- Проверьте названия колонок (должны совпадать с рекомендумыми)
- Убедитесь что данные в правильном формате
- Проверьте лог ошибок в консоли

### Неправильно определяется роль
- Добавьте ключевые слова в функцию `map_role_id()` в `import_resumes.py`
- Или укажите роль вручную в файле данных

## API Endpoint

Для программного импорта:

```bash
POST /api/import-resumes
Content-Type: multipart/form-data

file: <binary>
```

**Ответ:**
```json
{
  "success": true,
  "imported": 95,
  "total": 100,
  "message": "Импортировано 95 из 100 резюме"
}
```
