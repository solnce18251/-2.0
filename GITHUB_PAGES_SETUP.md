# 🚀 Как включить GitHub Pages

## Пошаговая инструкция

### Шаг 1: Откройте настройки репозитория

1. Перейдите на страницу репозитория:
   ```
   https://github.com/solnce18251/-2.0
   ```

2. Нажмите на вкладку **Settings** (⚙️) вверху

---

### Шаг 2: Перейдите в раздел Pages

В левом меню найдите и нажмите **Pages**

Или откройте напрямую:
```
https://github.com/solnce18251/-2.0/settings/pages
```

---

### Шаг 3: Настройте источник

В разделе **Build and deployment**:

| Настройка | Выберите |
|-----------|----------|
| **Source** | `Deploy from a branch` |
| **Branch** | `main` |
| **Folder** | `/docs` |

**Визуально:**

```
┌─────────────────────────────────────────┐
│ Build and deployment                    │
├─────────────────────────────────────────┤
│ Source:  [Deploy from a branch ▼]      │
│                                          │
│ Branch:  [main ▼]        Folder: [/docs ▼] │
│                                          │
│              [Save]                      │
└─────────────────────────────────────────┘
```

---

### Шаг 4: Сохраните

Нажмите кнопку **Save**

---

### Шаг 5: Подождите публикацию

GitHub начнёт обработку:

```
┌─────────────────────────────────────────┐
│ GitHub Pages                            │
├─────────────────────────────────────────┤
│ Your site is live at:                   │
│ 🎉 https://solnce18251.github.io/-2.0/ │
└─────────────────────────────────────────┘
```

Обычно занимает **1-3 минуты**.

## ⚠️ Важные заметки

### Демо-режим

На GitHub Pages работает **только интерфейс**:
- ✅ Фильтры и графики работают
- ✅ Дизайн и стили работают
- ❌ Парсинг не работает
- ❌ Импорт из файлов не работает
- ❌ Данные моковые (примерные)

### Для полноценной работы:

Используйте **Render** или **Railway**:

```bash
# Render (рекомендуется)
https://render.com/deploy?repo=https://github.com/solnce18251/-2.0

# Railway
https://railway.app
```

## 🔄 Обновление сайта

После изменений в коде:

```bash
# Сгенерировать статическую версию
python generate_static.py

# Закоммитить и отправить
git add docs
git commit -m "docs: update static version"
git push
```

GitHub Pages обновится автоматически через 1-2 минуты.

## 🆘 Troubleshooting

### Страница 404

**Проблема:** Страница не найдена

**Решение:**
1. Подождите 2-3 минуты после включения Pages
2. Проверьте что выбрана папка `/docs`
3. Очистите кэш браузера (Ctrl+Shift+R)

### Не применились изменения

**Проблема:** Сайт показывает старую версию

**Решение:**
1. Проверьте что изменения закоммичены
2. Проверьте что push выполнен
3. Посмотрите **Deployments** в репозитории
4. Очистите кэш браузера

### Не работают стили/скрипты

**Проблема:** Сайт выглядит сломанным

**Решение:**
1. Откройте консоль (F12)
2. Проверьте ошибки
3. Убедитесь что CDN доступны
4. Проверьте CORS политики
