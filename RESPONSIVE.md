# 📱 Адаптивность сайта

## Поддерживаемые устройства

### Мобильные телефоны

#### Маленькие (до 375px)
- iPhone SE, iPhone 5s
- Небольшие Android устройства

**Особенности:**
- 1 колонка в карточках статистики
- Компактный логотип (36x36px)
- Вертикальное расположение кнопок
- Уменьшенные шрифты

#### Стандартные (376px - 768px)
- iPhone 12/13/14, Samsung Galaxy S
- Большинство Android устройств

**Особенности:**
- 2 колонки в карточках статистики
- Логотип 40x40px
- Вертикальные кнопки
- Адаптивные графики

#### Планшеты (769px - 1024px)
- iPad, iPad Mini
- Samsung Galaxy Tab

**Особенности:**
- 2 колонки в статистике и графиках
- 2 колонки в фильтрах
- Горизонтальные графики

---

### Десктопы

#### Ноутбуки (1025px - 1440px)
- MacBook Air 13"
- Ноутбуки 13-15"

**Особенности:**
- 4 колонки в карточках статистики
- 2 колонки в графиках
- Оптимальная ширина контейнера

#### Большие мониторы (1441px+)
- iMac 27"
- Мониторы 24-27"

**Особенности:**
- Максимальная ширина 1400px
- 4 колонки в статистике
- 2 колонки в графиках

---

### Ориентация устройства

#### Портретная
- Вертикальное расположение элементов
- Кнопки на всю ширину

#### Пейзажная (мобильные)
- 4 колонки в статистике
- Компактные отступы

## Тестирование

### Chrome DevTools

1. Откройте сайт в Chrome
2. Нажмите `F12` или `Ctrl+Shift+I`
3. Нажмите `Ctrl+Shift+M` (Device Toolbar)
4. Выберите устройство:
   - iPhone 12 Pro (390x844)
   - iPad Air (820x1180)
   - Responsive (настраиваемый)

## Рекомендации

### Мобильные

✅ **Хорошо:**
- Крупные кнопки (легко нажимать)
- Минимум текста
- Вертикальная прокрутка
- Простые фильтры

❌ **Избегать:**
- Мелкий текст (< 14px)
- Горизонтальная прокрутка
- Сложные таблицы

### Планшеты

✅ **Хорошо:**
- 2 колонки для карточек
- Адаптивные графики
- Сенсорное управление

❌ **Избегать:**
- Слишком плотной компоновки
- Мелких кнопок

### Десктоп

✅ **Хорошо:**
- 4 колонки в статистике
- Полноэкранные графики
- Все фильтры видны

❌ **Избегать:**
- Слишком широких строк текста
- Пустого пространства

---

## Печать

При печати (`Ctrl+P`) скрываются:
- Header
- Фильтры
- Кнопки
- Модальные окна

Остаются:
- Заголовок
- Статистика
- Графики (если помещаются)

## Производительность

### Мобильные

## Breakpoints CSS

```css
/* Маленькие мобильные */
@media (max-width: 375px) { }

/* Мобильные */
@media (max-width: 768px) { }

/* Планшеты */
@media (min-width: 769px) and (max-width: 1024px) { }

/* Десктопы */
@media (min-width: 1025px) and (max-width: 1440px) { }

/* Большие экраны */
@media (min-width: 1441px) { }
```
